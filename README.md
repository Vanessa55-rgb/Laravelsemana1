# Laravelsemana1

Este proyecto es una aplicación web desarrollada con el framework **Laravel**, personalizada para gestionar tareas y usuarios de manera eficiente.

## Requisitos

Asegúrate de tener instalados los siguientes componentes:

*   **PHP** (>= 8.2)
*   **Composer**
*   **Node.js & NPM**
*   **Base de Datos** (MySQL, PostgreSQL, SQLite, etc.)

## Pasos para la Instalación

Sigue estos pasos para poner en marcha el proyecto:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/Vanessa55-rgb/Laravelsemana1.git
    cd Laravelsemana1
    ```

2.  **Instalar dependencias de PHP:**
    ```bash
    composer install
    ```

3.  **Configurar el entorno:**
    Crea tu archivo `.env` a partir del ejemplo:
    ```bash
    cp .env.example .env
    ```

4.  **Instalar dependencias de Node.js y compilar:**
    ```bash
    npm install
    npm run dev
    ```

5.  **Iniciar el servidor:**
    ```bash
    php artisan serve
    ```

## Stack Tecnológico

Este proyecto utiliza las siguientes herramientas principales:

*   **Laravel Jetstream**: Para el andamiaje de la aplicación y autenticación.
*   **Livewire**: Para construir interfaces dinámicas y reactivas.
*   **Tailwind CSS**: Para el diseño y estilos modernos.
